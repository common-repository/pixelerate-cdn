Routes image files through the Pixelerate CDN and optimization services.

== Description ==
The Pixelerate Optimization CDN takes your images and optimizes their size and delivery routes making the images and your website load quicker for your customers.

The Pixelerate Image CDN requires a valid subscription from Pixelerate.io in order to fully work.
For more information on how this works, please visit https://www.pixelerate.io/

= Features =
* Makes your images load from a CDN
* Optimizes Png and Jpeg images to smaller files
* Enables automatic optimization to WebP for better optimized images.
* Faster page loads and smaller images increases SEO
* HTTPS delivery of content

= System Requirements =
* PHP >=5.6
* WordPress >=4.6
